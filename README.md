##  Getting Started


###  Instalación

1. Clonar el repositorio:

```sh
git clone https://github.com/saturnette/PokemonRoulette-React
```

2. Cambiar al directorio del proyecto:

```sh
cd PokemonRoulette-React
```

3. Instalación de dependencias:

```sh
> npm i
```

4. Ejecución del proyecto:

```sh
> npm run dev
```
